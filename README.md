🛸 iMersFireship: Digital Marketplace & Affiliate System

iMersFireship adalah solusi penjualan produk digital (E-book, Kursus Online, Template) berbasis cloud yang menggabungkan kekuatan Google Sheets sebagai database dan Vercel sebagai frontend.

Sistem ini dilengkapi dengan fitur Dual-Layer Security yang mengunci database menggunakan lisensi resmi, sehingga aman dari penggandaan ilegal.

✨ Fitur Utama

Dashboard Modern: Tampilan mewah berbasis Tailwind CSS (Glassmorphism).

Sistem Afiliasi: Kode unik untuk tiap user dan dashboard komisi otomatis.

WhatsApp Gateway: Notifikasi otomatis via Fonnte/Starsender.

Smart Sync Database: Update struktur tabel tanpa menghapus data kustomer.

LMS & Download: Dukungan materi video atau link download file langsung.

🚀 Panduan Instalasi Cepat

1. Persiapan Backend (Google Sheets)

Salin (Make a copy) file Google Sheet iMersFireship.

Buka Extensions > Apps Script.

Klik Deploy > New Deployment.

Pilih Web App, set akses ke Anyone, lalu klik Deploy.

Salin URL Web App yang muncul (Berakhir dengan /exec).

2. Konfigurasi Frontend (index.html)

Buka file index.html di repository ini, lalu cari baris 1334.
Ganti tulisan URL_WEB_APP_APPS_SCRIPT_LO_DISINI dengan URL yang Anda salin tadi.

3. Deploy ke Vercel

Hubungkan repository GitHub Anda ke Vercel.

Tunggu proses build selesai.

Buka dashboard Vercel Anda dan lakukan aktivasi menggunakan lisensi.

🛡️ Lisensi & Keamanan

Aplikasi ini dilindungi oleh sistem lisensi Master Library. Penggunaan tanpa lisensi yang valid akan menyebabkan akses database terputus otomatis.
